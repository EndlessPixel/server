XMCL 启动器(x-minecraft-launcher)

该启动器具备以下特性：

  • 📥 下载并自动完成。支持从官方或第三方镜像下载 Minecraft Forge、Fabric、Quilt、OptiFine 以及 JVM。
  • ⚡️ 快速下载。通过 HTTP/HTTPS 代理复用 socket，并发分批下载文件。
  • 💻 跨平台。该启动器基于 Electron，支持 🗔 Windows 10/11、🍎 MacOS 和 🐧 Linux。
  • 📚 多实例。用户可以创建多个实例来隔离不同的版本、mod 和启动设置。
  • 🗂 管理所有资源。使用（硬/符号）链接在实例中安装资源，保持磁盘使用率最佳，避免 Mod 副本冗余。
  • 🔥 内置支持 CurseForge、Modrinth。您可以在 Launcher 中下载资源。
  • 📦 支持导入/导出 CurseForge 和 Modrinth 模组包，符合相关要求。
  • 🔒 支持多个账户系统。内置 Microsoft 登录和 Mojang Yggdrasil API，还支持 ely.by 和 littleskin.cn，同时可添加第三方身份验证服务器。
  • 🔗 用户之间的点对点连接。即使不在同一个物理 LAN 中，也能通过 LAN 玩多人游戏。
  • 🔑 代码标志和现代包装。在 Windows 下安装应用程序，不会再收到来自浏览器的阻止消息或看到 SmartScreen 错误。
