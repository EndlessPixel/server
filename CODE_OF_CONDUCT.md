# 行为准则
## 我们的承诺
为了营造开放、包容的社区环境，我们承诺：
- 不论年龄、体型、残障、种族、性别认同、性取向、经验、教育、社会经济地位、国籍、宗教等，所有参与者都能受到尊重
- 杜绝任何形式的骚扰、歧视行为
- 保持专业、友好的沟通氛围

## 我们的准则
社区成员应遵守的行为规范：
1. 使用友好、包容的语言
2. 尊重不同的观点和经验
3. 接受建设性批评并礼貌回应
4. 关注社区整体利益，而非个人得失
5. 对其他社区成员保持同理心

## 不被允许的行为
1. 使用性化的语言/图像，以及不必要的性暗示
2. 人身攻击、辱骂、歧视性言论
3. 公开/私下骚扰
4. 未经许可公开他人隐私（如邮箱、地址）
5. 其他违反公序良俗/专业精神的行为

## 责任与执行
### 维护者责任
- 明确可接受行为的标准
- 对违规行为采取公平、及时的纠正措施
- 对违规行为的处理结果进行记录（私密）

### 违规处理流程
1. 首次违规：私下警告，要求纠正行为
2. 再次违规：临时禁止参与社区活动（7-30天）
3. 严重/多次违规：永久禁止参与本项目所有社区活动

### 举报方式
- 邮件：support@endlesspixel.com
- GitHub：联系仓库维护者（@EndlessPixel 核心团队）

## 适用范围
本准则适用于所有项目相关场景：
- GitHub Issues/Pull Requests/Discussions
- 项目相关的聊天群组/邮件列表
- 线下活动（如有）

## 准则修订
本准则可能会根据社区反馈修订，修订后会在仓库中通知。
